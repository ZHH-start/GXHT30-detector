/*********************************************************************************************/
【*】程序简介

-工程名称：systick实现us级延时，且兼容HAL_Delay
-实验平台: 野火 ebf_py32f002b PY32开发板 

-功能介绍：控制开发板上的LED2灯1s闪烁。

/*********************************************************************************************/
【*】实验操作：
编译并下载程序到开发板，呈现LED2灯1s闪烁

【*】注意事项：
 串口波特率：115200

/*********************************************************************************************/
【*】引脚分配

LED灯：
共阳极，低电平亮。
	LED2引脚为PA1
	LED3引脚为PA5
	LED4引脚为PA4
	


KEY:
两个按键均有硬件去抖，按下的时候均为低电平，不按时为高电平。
	KEY1<--->PA0
	KEY2<--->PA3

USART：
	CH340  RX  <----->  USART0  TX  (PB4)
	CH340  TX  <----->  USART0  RX  (PB5)			
		
/*********************************************************************************************/		
【*】程序描述：
<main.c>
	1.初始化LED和按键中断
	2.按键中断，若按键按下则反转LED

< bsp_systick.c >

兼容HAL_Delay（）函数，可以与其公用。
	
-ms延时程序

	调用us延时程序实现

-us延时程序
	1.获取Systick->LOAD寄存器值
	2.计算延时节拍数
	3.等待S时间大于所需延时就退出
